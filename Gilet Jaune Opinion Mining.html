Explication du problème¶

Les "gilets jaunes" est le nom donn au mouvement dorigine citoyenne de contestation contre la hausse des taxes. Nous nous intressons dans ce travail linfluence croise de la science des donnes et des sciences. Sociales du politique. Le mouvement s'largit de plus en plus avec des dommages encore plus important.

Présentation de la solution proposée¶

Ce mouvement est apparu principalement sur les rseaux sociaux, il est donc trs important de voir les avis des internautes a ce sujet. Dans notre cas, nous allons tudier l'opinion des internautes sur Tweeter.

def cleanTxt(text):
    text = re.sub(r'@[A-Za-z0-9_ :]+', '', text)
    text = re.sub(r'#', '', text)
    text = re.sub(r'RT[\s]+', '', text)
    text = re.sub(r'https?:\/\/\+S', '', text)
    text = re.sub(r'https', '', text)
    text = re.sub(r':', '', text)
    text = re.sub(r'[0-9]', '', text)
    text = text.lower()
    return text
